About me

The person behind the portfolio

Hi, I'm Abdul — a student splitting my time between two things I enjoy: building networks and building web pages. On one side I'm working through Cisco Packet Tracer labs covering VLANs, ACLs, routing and firewall configuration. On the other, I'm learning front-end development, from plain HTML and CSS through to JavaScript interactivity.

I like the way both fields reward the same habits: reading error messages carefully, testing one change at a time, and documenting what actually fixed the problem. Whether that's a misconfigured VLAN or a CSS property that isn't valid, the debugging mindset is the same.

Outside of coursework, I'm currently building learning resources to help my brother pick up HTML — teaching something is usually the fastest way to actually understand it yourself.

I'm currently looking for internship or entry-level opportunities in web development or network administration.
